# MintSchedule

MintSchedule is a minimalist smart calendar app designed to simplify daily planning with a clean, calm interface.

Beyond basic scheduling, MintSchedule is built around the idea of **smart automation**: events and routines can be used to trigger actions (for example, turning on lights or powering devices at a specific time) by syncing with smart home platforms and automation services.

## Project goals
- Simple, distraction-free calendar experience  
- Fast event creation and day-to-day planning  
- Optional automations tied to schedules (smart home / routines)
